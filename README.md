🍱 FoodBridge – Community Food Sharing Platform
🌍 Problem Statement

Food waste is a major global issue, while many people struggle to access daily meals.
FoodBridge is a community-driven platform that connects food donors with people in need, reducing food waste and fighting hunger.

The platform allows users to:

🥘 Post surplus food details

📍 Share pickup location

🙋 Request available food

🔁 Connect donors and receivers safely

🚀 Features (MVP)
👤 User Management

Register as Donor or Receiver

Login authentication

Role-based access

🍱 Food Posting

Add food name

Quantity

Expiry time

Pickup location

Description

Contact info (shared after acceptance)

🔎 Browse Available Food

View all active food posts

Filter by location

Auto-hide expired food

📩 Request System

Receiver can request food

Donor can Accept / Reject

Contact details shared only after acceptance
